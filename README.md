# TODO List ✔️

Example TODO List application.

## Dependencies

* NuxtJS
* Typescript
* Buefy CSS
* Tailwind CSS
* SASS
* Pinia Storage
* Cypress
* Jest

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
